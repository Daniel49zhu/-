- P1 向量(Vector)
    
    - 物理学：向量是空间中的一个箭头，有长度和方向
    - 计算机：向量是有序的数字列表
    - 数学：向量是任意有大小和方向的量
    
    ![二维向量](images/1.jpg "二维向量")
    
    从原点出发，第一个数代表要顺着x轴走多远，第二个数代表要顺着y轴走多远
    
    ![三维向量](images/2.jpg "三维向量")
    
     从原点出发，第一个数代表要顺着x轴走多远，第二个数代表要顺着y轴走多远，第三个数代表沿着z轴的距离
