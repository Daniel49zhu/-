#### 注解

   注解也被称为元数据，为我们在代码中添加信息提供了一个形式化的方式。一定程度上把元数据和
   源代码结合在一起。Jdk5引入了注解，并内置了三种注解
   - @Override 表示当前方法覆盖超类中的方法
   - @Deprecated 表示当前方法已被废弃，如果使用了会在编译期间发出警告信息
   - @SuppressWarnings 关闭不当的编译器警告信息
   
   另外还提供了四种注解负责新注解的创建，会在后面学习。
